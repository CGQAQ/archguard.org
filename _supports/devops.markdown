## DevOps

DevOps（英文 Development 和 Operations 的组合）是一组过程、方法与系统的统称，用于促进开发（应用程序/软件工程）、技术运营和质量保障（QA）部门之间的沟通、协作与整合。里面很强调的一点是实现端到端的持续价值交付


## DevOps能力提升
很多人觉得DevOps会牵涉到很多工具的使用，当在一个大型组织要规模化的提升DevOps能力的时候，通常需要关注几点：

* 组织规范
* 平台支撑
* 实践固化
* 人才培养
